# 60 Module Handoffs

## PolicyGraph
Build a real node and transition schema for fulfillment exceptions.

## ClarificationRouter
Build missing-slot detection and pre-action clarification logic.

## ActionGating
Build an allow or deny layer for risky actions.

## Eval Factory
Build trace replay, scoring, and error reporting.

## Experience Loop
Build experience extraction and strategy-memory update flow.

## Simulator
Build synthetic cases and perturbation injection.

## Rule for all implementers
Every module must define:
- inputs
- outputs
- failure cases
- trace fields
- minimal evaluation cases
