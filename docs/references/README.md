# Reference Pack

本目录不继续写业务代码，而是为论文与后续强模型协同开发提供“牢固基础”。

## 目录说明
- `01-literature-matrix.md`：核心论文与官方工程文章矩阵。
- `02-module-to-literature.md`：仓库模块与论文/工程趋势映射。
- `03-reproduction-chain.md`：从论文到系统原型的复现链路。
- `04-public-assets.md`：公开可用代码、数据、文章与缺失项说明。
- `05-agent-thesis-context-pack.md`：给后续更强模型使用的完整上下文包。
- `references.bib`：开题/写作可直接复用的 Bib 条目。
- `../handoffs/module-development-briefs.md`：按模块拆分的开发 brief。

## 使用原则
1. 只引用公开可访问的论文、官方工程文章和公开仓库。
2. 对于未公开源码的论文，只保留：结论、方法摘要、复现建议、缺失项说明。
3. 不把第三方论文直接翻译成代码，而是转成：系统需求、模块规格、伪代码、评估口径。
4. 所有“持续进化”相关内容优先走非参数路径：经验回流、策略记忆、澄清策略优化、评估反馈。
