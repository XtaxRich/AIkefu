# 20 Literature Matrix

## Core papers and articles
- JourneyBench / Beyond IVR: customer support should be judged by business adherence.
- Process Evaluation for Agentic Systems: evaluation must inspect full traces.
- Scaling Intent Understanding with Clarification Questions: clarification is a first-class decision.
- SAGE: simulators help find agent failures.
- Memory benchmark for LLM agents: memory must be evaluated as a system layer.
- Agentic Context Engineering: feedback can improve context and strategy.
- CLIN: continual improvement can happen without parameter updates.
- AMIE: self-play and automated feedback can improve multi-turn systems.
- Anthropic engineering articles on agents, context, tools, harnesses, and auto mode.
- OpenAI evals for agent skills.

## How they map to this thesis
- policy adherence -> service execution objective
- trace evaluation -> audit and metrics
- clarification -> pre-action control
- simulator -> offline testing and regression
- memory / ACE / CLIN -> continual improvement
- harness articles -> runtime design
