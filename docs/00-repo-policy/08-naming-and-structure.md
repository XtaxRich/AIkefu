# 08 Naming and Structure

## Directory rule
Use:
- two-digit prefix
- clear topic name

Examples:
- `00-repo-policy`
- `30-architecture`
- `50-experiments`

## File rule
Use:
- two-digit prefix
- one topic per file
- no vague names

Examples:
- `01-branch-and-review.md`
- `02-code-admission.md`

## README rule
README is for overview and navigation only.
Detailed content should go into numbered files.

## Avoid
- `tmp.md`
- `final_final.md`
- `misc/`
- unclear test file names
