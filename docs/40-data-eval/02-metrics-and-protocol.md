# 02 Metrics and Protocol

## Core metrics
- policy_adherence
- required_step_coverage
- clarification_usefulness
- action_order_correctness
- escalation_correctness
- recovery_success
- cost_per_success_case

## Eval rule
不能只看 final answer，必须看完整 trace。

## Minimum scoring
- path score
- clarification score
- action-order score
- escalation score
- overall pass/fail
